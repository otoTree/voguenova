# 产品需求文档 (PRD): VogueNova AI 模特 IP 平台

## 1. 产品愿景
打造一个全链路的 AI 模特 IP 孵化与展示平台。用户可以通过 AI 技术从零到一生成拥有独特人设的虚拟模特，并为其进行高级时尚样片的 AI 拍摄与动态视频生成，最终形成完整的虚拟时尚偶像 IP。整体视觉对标香奈儿 (Chanel)，呈现极简、优雅、高级的黑白金时尚质感。

## 2. 核心功能模块

### 2.1 模特 IP 档案构建 (Text Generation)
- **人设生成**：基于 `gpt-4o`，生成模特的背景故事、性格特征、外貌描述、穿搭风格。
- **档案管理**：保存模特的基本信息、生成提示词 (Prompt) 模板。

### 2.2 AI 时尚大片拍摄 (Image Generation)
- **文生图/图生图棚拍**：基于 `gemini-3-pro-image-preview`，用户输入场景需求（或上传参考图），结合模特的基础 Prompt，生成高质量的纯色背景棚拍图、外景街拍等。
- **画廊管理**：模特专属的照片墙，支持瀑布流展示与高级灯箱预览。

### 2.3 动态视频生成与队列 (Video Generation)
- **智能分镜视频**：基于 `kling-v3-omni-pro`，为模特照片生成动态走秀或展示视频。
- **异步队列管理**：由于视频生成耗时较长，系统采用 Redis 队列管理视频生成任务，支持轮询状态查询与前端进度条展示。

## 3. 技术架构与选型
- **前端框架**：Next.js (App Router)
- **状态管理**：Zustand
- **UI 组件库**：Shadcn-ui (完整引入，定制黑白高级主题)
- **样式方案**：Tailwind CSS
- **数据库**：PostgreSQL (直接使用 `pg` 驱动，不使用 Prisma)
- **缓存与消息队列**：Redis (用于视频生成任务队列管理与状态轮询)
- **AI 接口驱动**：封装 Cloubic API 客户端（包含文本、图像、视频服务）

## 4. UI/UX 设计规范 (香奈儿风格)
- **主色调**：
  - 纯黑 (`#000000`) 与 纯白 (`#FFFFFF`) 构建极致对比。
  - 点缀色：淡雅香槟金 (`#D4AF37`) 用于强调按钮或重要状态。
- **排版**：
  - 大量留白 (Negative Space)，克制的边框与阴影。
  - 字体：使用优雅的无衬线体与经典的衬线体搭配（如 Playfair Display）。
- **交互**：
  - 丝滑的淡入淡出动画，拒绝浮夸特效，追求物理质感与克制。

## 5. 核心业务流程
1. **创建 IP**：填写基础词条 -> 调用 Text API 生成完整档案 -> 存入 PostgreSQL。
2. **生成样片**：选择 IP -> 编写拍摄场景 -> 调用 Image API -> 返回图片 URL -> 存入 PostgreSQL。
3. **视频化**：选择样片 -> 提交视频生成任务 -> 写入 Redis 队列 -> 调用 Video API -> 轮询 Task ID -> 视频就绪后更新 PostgreSQL。
